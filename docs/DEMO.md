# Demo (Chatwoot en vivo)

## Cómo probar
1) Abrir el widget de Chatwoot (Inbox “Laburen Challenge”).
2) Enviar: “Quiero ver productos”
3) Enviar: “Agregar ID 2 cantidad 2”
4) Enviar: “Ver carrito”
5) Enviar: “Modificar ID 2 cantidad 1”
6) Enviar: “Finalizar compra”
